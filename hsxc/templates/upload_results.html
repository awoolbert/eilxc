Upload a Results File

Upload a txt file with the race results. If the race has an official name, include it in the first row of the file. The date of the race should also be included in the first few ross.

The individual results rows should include the following data:

Data Description
Place the overall place earned by runner
Bib Number (optional, may not come before place) runner's bib number
First Name runner's first name
Last Name runner's last name
Class/Grade (optional, many not come before name) year of runner's graduation (e.g., 2025), or grade (e.g., 11)
Time runner's time in MIN:SEC format (e.g., 21:34 or 21:34.5)
School (must be separated from Name by Time or Class) runner's school (e.g., Newton Country Day School or NCDS)
